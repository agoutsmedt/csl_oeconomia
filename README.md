# csl_oeconomia

This repository contains Citation Style Language (CSL) files for formatting bibliographies in [OEconomia](https://oeconomia.revues.org/) journal articles.

## Files

The repository contains two CSL files:

- **oeconomia_en.csl**: Citation style for English articles
- **oeconomia_fr.csl**: Citation style for French articles

## Purpose

This repository serves two main purposes:

1. **Bibliography Formatting**: Provides standardized CSL files to format bibliographies and citations for OEconomia journal articles, ensuring consistent citation formatting across publications.

2. **Issue Tracking**: Serves as a centralized location to report and track formatting problems or inconsistencies in the citation styles. If you encounter any issues with the bibliography formatting, please open an issue in this repository.

## Usage

To use these citation styles:

1. Download the appropriate CSL file (`oeconomia_en.csl` for English or `oeconomia_fr.csl` for French)
2. Import it into your reference management software (Zotero, Mendeley, etc.)
3. Apply the style to your document

## Contributing

If you notice any formatting issues or have suggestions for improvements, please [open an issue](../../issues) to help us maintain accurate citation styles for OEconomia.
